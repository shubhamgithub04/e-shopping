## Run Locallygit init


Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd e-shopping
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
* [Bootstrap](https://getbootstrap.com/)
* [Fake Store API](https://fakestoreapi.com/)
